Задание

Дана длинная строка с различными символами ASCII, в ней необходимо заменить все
цифры «1» на буквы «Q», а все буквы «Q» - на цифры «1». Для решения задачи
разрешено заменять во всём тексте один произвольный набор символов на другой любое
количество раз. Например, в результате замены «ww»->«wert» строка «wwwwwQ»
перейдёт в строку «wertwertwQ». Напишите программу, которая генерирует тестовую
строку и производит необходимые замены оптимальным образом (за минимальное
количество замен).

Входные данные

Сгенерированная строка случайной длинны состоящая из случайных символов ASCII.

Выходные данные

Строка, в которой все «Q» заменены на «1», а все «1» заменены на «Q».
